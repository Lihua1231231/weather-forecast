## 项目概览

### 这是什么？

这是您的个人主页网站，用于向HR展示您的教育背景、实习经历、技能特长等信息。网站包含一个AI对话功能，可以24小时在线回答访问者关于您的问题。

### 技术栈

- **前端框架**：React（使用Vite构建工具）
- **编程语言**：TypeScript
- **样式**：Tailwind CSS
- **AI服务**：硅基流动（SiliconFlow）的DeepSeek-V3模型

**重要提示：您不需要懂编程！** 本文档将教您如何轻松修改所有个人信息。

## 文件夹结构说明

```
向金涛的个人主页/
├── src/                    【源代码文件夹】
│   ├── portfolioData.ts    ⭐ 最重要！所有个人信息都在这里
│   ├── App.tsx             网站主页面（一般不需要改）
│   ├── components/         组件文件夹（一般不需要改）
│   └── ...
├── public/                 【静态资源文件夹】
│   ├── photo1.jpg          您的第一张照片
│   ├── photo2.jpg          您的第二张照片
│   ├── photo3.jpg          您的第三张照片
│   └── resume.pdf          您的简历PDF
├── dist/                   【生产版本文件夹】⭐ 部署时上传这个文件夹
├── node_modules/           依赖包（不要动）
├── package.json            项目配置（不要动）
└── ...
```

### 核心文件说明

#### 1. `/src/portfolioData.ts` ⭐⭐⭐

**这是最最重要的文件！** 所有您的个人信息、AI配置都在这个文件里。

#### 2. `/public/` 文件夹

存放照片和简历PDF。替换文件时，**文件名必须保持一致**（photo1.jpg、 photo2.jpg、photo3.jpg、resume.pdf）。

#### 3. `/dist/` 文件夹

这是网站的生产版本，上传到Netlify时就用这个文件夹。

## 如何修改个人信息

### 步骤1：找到关键文件

用任何文本编辑器（推荐使用**记事本++**或**VS Code**）打开文件：

```
/Users/wepie/Desktop/文件夹/向金涛的个人主页/src/portfolioData.ts
```

### 步骤2：修改个人信息

在这个文件中，您会看到清晰的结构：

```typescript
export const portfolioData = {
  // ========== 1. 基本信息 ==========
  name: "向金涛",              // 修改这里可以改名字
  tagline: "积极主动 | 稳重可靠 | 善于思考",  // 修改标语
  
  // ========== 2. 照片轮播 ==========
  photos: [
    "/photo1.jpg",  // 对应 public/photo1.jpg
    "/photo2.jpg",  // 对应 public/photo2.jpg
    "/photo3.jpg",  // 对应 public/photo3.jpg
  ],
  
  // ========== 3. 简历文件 ==========
  resumeUrl: "/resume.pdf",  // 对应 public/resume.pdf
  
  // ========== 4. 关于我 ==========
  about: {
    mbti: "ISFJ",  // 修改MBTI类型
    mbtiDescription: "可靠尽责...",  // 修改描述
    hobbies: [
      { label: "健身", icon: "💪" },
      // 可以添加更多爱好
    ]
  },
  
  // ... 还有更多配置
}
```

**修改注意事项：**
- ⚠️ 不要删除引号`""`、逗号`,`、冒号`:`等符号
- ⚠️ 修改中文内容时保持在引号内
- ⚠️ 保存时使用UTF-8编码

### 步骤3：更新照片和简历

1. 准备好您的照片（JPG格式）和简历（PDF格式）
2. 将文件命名为：`photo1.jpg`、`photo2.jpg`、`photo3.jpg`、`resume.pdf`
3. 替换`/public/`文件夹中的对应文件

**照片建议：**
- 尺寸：建议 600x400 像素或更高
- 格式：JPG
- 大小：每张不超过2MB

## 如何更新AI知识库

AI知识库决定了AI分身能回答哪些问题。所有知识都在`portfolioData.ts`文件的`systemPrompt`部分。

### 在哪里修改？

打开`/src/portfolioData.ts`，找到这一部分：

```typescript
ai: {
  apiKey: "sk-bvbrgrrczqstuxjlvbyfxwnkcmfkuqjeapvdeylbvlczsnoh",
  model: "deepseek-ai/DeepSeek-V3",
  apiEndpoint: "https://api.siliconflow.cn/v1/chat/completions",
  
  systemPrompt: `我是向金涛。请注意：你就是我本人，不是我的助手。
  
  ## 核心原则（必须严格遵守）
  ...
  
  # 完整知识库
  ...
  `,
}
```

### 如何添加知识？

在`systemPrompt`的"完整知识库"部分，按照现有格式添加内容：

**示例1：添加新的实习经历**

```markdown
## 四、实习经历

...现有内容...

## 4. XX公司（2026.03-2026.06）
**职位**：XX实习生

- 负责XX工作
- 完成XX项目
- 获得XX成果
```

**示例2：添加新的技能**

```markdown
## 七、技能

- **计算机**：熟练使用...
- **数据分析**：掌握...
- **新技能**：会使用XXX  // 在这里添加
```

**重要提示：**
- ✅ 必须使用Markdown格式（`##`表示标题，`-`表示列表）
- ✅ 保持与现有内容相同的格式和缩进
- ✅ 所有新内容写在反引号`` ` ``之内
- ⚠️ 不要修改其他配置（API key、model等）

## 如何部署到Netlify

### 前提条件

每次修改`portfolioData.ts`后，都需要重新构建生产版本。

### 步骤1：构建生产版本

1. 打开终端（Mac系统打开"终端"应用）
2. 进入项目文件夹：
   ```bash
   cd "/Users/wepie/Desktop/文件夹/向金涛的个人主页"
   ```
3. 运行构建命令：
   ```bash
   npm run build
   ```
4. 等待命令执行完成（通常10-30秒）
5. 构建成功后，`dist/`文件夹会更新

### 步骤2：上传到Netlify

#### 方法一：拖拽上传（最简单）

1. 打开浏览器，访问：https://app.netlify.com/
2. 登录您的Netlify账号
3. 找到您的网站项目 或 点击"Add new site" → "Deploy manually"
4. **直接拖拽`dist`文件夹** 到页面上传区域
5. 等待部署完成（通常1-2分钟）
6. 部署成功后，您会看到网站URL

#### 方法二：使用Netlify CLI（略复杂）

如果您熟悉命令行，也可以使用：

```bash
# 1. 安装Netlify CLI（只需一次）
npm install -g netlify-cli

# 2. 登录Netlify
netlify login

# 3. 部署
netlify deploy --prod --dir=dist
```

### 步骤3：验证部署

1. 访问Netlify给您的网站URL
2. 测试AI对话功能，问几个问题确保知识库正常
3. 检查照片和简历能否正常下载

## 常见问题解答

### Q1: 我不小心删除了某些代码，怎么恢复？

**答**：如果您有GitHub或其他版本控制，可以恢复历史版本。如果没有，可以重新从备份恢复文件。建议每次修改前先备份整个项目文件夹。

### Q2: 修改后网站显示错误，怎么办？

**答**：
1. 检查`portfolioData.ts`文件中是否有语法错误（如缺少逗号、引号等）
2. 确保所有字符串都在引号内
3. 如果实在找不到问题，可以恢复到修改前的版本

### Q3: AI回答的内容不准确，怎么改？

**答**：在`portfolioData.ts`的`systemPrompt`部分修改或添加相关知识。记住：AI只能根据knowledge库回答，所以您需要把希望AI知道的信息都写进去。

### Q4: 如何更换照片？

**答**：
1. 准备好新照片，命名为`photo1.jpg`、`photo2.jpg`、`photo3.jpg`
2. 替换`/public/`文件夹中的对应文件
3. 重新构建：`npm run build`
4. 重新部署到Netlify

### Q5: API key泄露了怎么办？

**答**：
1. 立即登录硅基流动平台：https://cloud.siliconflow.cn/
2. 在API管理页面删除旧的API key
3. 生成新的API key
4. 在`portfolioData.ts`中更新`apiKey`字段
5. 重新构建和部署

### Q6: 如何修改AI模型？

**答**：不建议修改。当前使用的DeepSeek-V3是2025年最先进的对话模型。如果确实需要更换，修改`portfolioData.ts`中的`model`字段，查看硅基流动支持的模型列表。

### Q7: 网站访问速度慢怎么办？

**答**：
1. 优化照片大小（压缩图片，建议单张不超过500KB）
2. 使用Netlify的CDN加速已经默认开启
3. 确保简历PDF文件不超过5MB

### Q8: 我想添加新的板块怎么办？

**答**：这需要修改`App.tsx`文件，涉及React开发知识。建议您联系开发者协助，或者学习React基础后再进行修改。
